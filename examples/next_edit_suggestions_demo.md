# Demo práctico: Next Edit Suggestions

Objetivo: demostrar sugerencias contextuales de edición en un archivo y cómo aplicarlas rápidamente.

Pre-requisitos
- Python (`py`) instalado.

Flujo de la demo (4–6 minutos)

1) Mostrar archivo original (ejemplo incluido en el script).
2) Ejecutar el script que genera 2 sugerencias de edición y muestra un diff previo/post.

```powershell
py .\examples\next_edit_suggestions_demo.py
```

3) Aplicar sugerencias: el script preguntará si deseas aplicar cada sugerencia.
4) Mostrar resultado final y explicar reglas que usaste para generar las sugerencias (claridad, tono, seguridad).

Material incluido
- `examples/next_edit_suggestions_demo.py` — script que crea sugerencias simuladas y permite aplicarlas.
