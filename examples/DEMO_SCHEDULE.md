# Programa sugerido para sesión de demos

Duración total sugerida: 75–100 minutos (ajustable).

Formato sugerido: 5–8 minutos por demo, con 1–2 minutos de transición.

Agenda recomendada

- 00:00 - 00:05 Introducción y objetivos de la sesión
- 00:05 - 00:13 Premium Requests (8 min)
- 00:13 - 00:20 Choosing the right AI model (7 min)
- 00:20 - 00:26 Modos de chat (6 min)
- 00:26 - 00:32 Ask (RAG) (6 min)
- 00:32 - 00:37 Edits (5 min)
- 00:37 - 00:43 Agent (6 min)
- 00:43 - 00:48 Plan - Insiders (5 min)
- 00:48 - 00:53 Next Edit Suggestions (5 min)
- 00:53 - 00:59 Code Review (6 min)
- 00:59 - 01:03 Prompt Files (4 min)
- 01:03 - 01:07 Custom Chat Modes (4 min)
- 01:07 - 01:12 Copilot en GitHub (5 min)
- 01:12 - 01:18 MCP (6 min)
- 01:18 - 01:25 Preguntas y cierre (7 min)

Notas:
- Ajusta tiempos según el público y profundidad técnica. Para público no técnico reduce demos a 4–6 demos clave.
- Deja un tiempo final para preguntas y feedback.
