# Demo práctico: Edits

Objetivo: demostrar cómo realizar ediciones de texto (mejorar claridad, formalizar, hacer más conciso) usando instrucciones de edición.

Pre-requisitos
- Python (`py`) instalado para ejecutar el script de ejemplo.

Flujo de la demo (5–8 minutos)

1) Mostrar el texto original (30s).

2) Ejecutar el script que aplica dos ediciones automáticas: "Formalizar" y "Hacer conciso".

```powershell
py .\examples\edits_demo.py
```

3) Comparar las versiones y explicar las decisiones de edición (qué se cambió y por qué).

4) Discutir parámetros que afectan las ediciones: temperatura (si aplica), instrucciones de estilo y limitaciones.

Material incluido
- `examples/edits_demo.py` — script que muestra el original y dos ediciones automatizadas (simuladas).
