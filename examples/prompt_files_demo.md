# Demo práctico: Prompt Files

Objetivo: mostrar cómo organizar prompts en archivos y utilizarlos desde scripts.

Flujo
1) Revisar el archivo de template `prompts/prompt_files_demo.md`.
2) Ejecutar el script:

```powershell
py .\examples\prompt_files_demo.py
```

El script cargará el template, reemplazará variables (`SERVICE_NAME`, `OWNER`) y mostrará el prompt final listo para enviar a la API.
