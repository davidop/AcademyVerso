# Prompt: Generar tests pytest para un módulo

Instrucciones para el asistente:

- Genera tests `pytest` para el módulo indicado, cubriendo casos de éxito, fallos y condiciones de borde.

Prompt:
```
Genera tests pytest para el módulo `auth.py` que tiene la función `authenticate(username, password)`. Asegúrate de cubrir credenciales válidas, contraseñas incorrectas y usuarios inexistentes.
```
